
# Port Scanner Using Python

## Overview
This project implements a Python-based TCP port scanner that identifies open ports
on a target host. It demonstrates basic network security concepts related to service
discovery and attack surface analysis.

## Features
- TCP socket-based port scanning
- Scans common service ports
- Clear command-line output
- Beginner-friendly and extensible

## Technologies Used
- Python 3
- socket

## Usage
```bash
python main.py <target_ip>
```

## Example
```bash
python main.py 192.168.0.1
```

## Disclaimer
Scan only systems you own or have permission to test.
